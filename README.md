
# 🎯 Number Guessing Game (Python)

A simple console-based Python game where the user tries to guess a randomly generated number between 1 and 50.

## 🧠 Description
The computer randomly selects a number between 1 and 50. The player has to guess it, and the program gives hints like "Too low!" or "Too high!" until the correct number is found.

## 🎮 How to Play
- Run the program
- Enter your guess
- Get hints after each attempt
- Keep guessing until you win 🎉

## 🛠️ Technologies Used
- Python 3
- random module
- while loop
- if-else conditions
- input handling
